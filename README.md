# AutoCodeRover
AutoCodeRover is an AI-based system for automated bug detection and fixing.

## Features
- Uses Large Language Models (LLMs)
- AST-based code analysis
- Fault localization using SBFL
- Automated patch generation

## Tech Stack
- Python
- LLMs (GPT)
- AST
- SBFL
- Docker
